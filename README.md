## ✨ Features

- 🤖 Use different chatbots in one app, currently supporting ChatGPT, new Bing Chat, Google Bard, Claude, and open-source models including LLama2, Vicuna, ChatGLM etc
- 💬 Chat with multiple chatbots at the same time, making it easy to compare their answers
- 🚀 Support ChatGPT API and GPT-4 Browsing
- 🔍 Shortcut to quickly activate the app anywhere in the browser
- 🎨 Markdown and code highlight support
- 📚 Prompt Library for custom prompts and community prompts
- 💾 Conversation history saved locally
- 📥 Export and Import all your data
- 🔗 Share conversation to markdown
- 🌙 Dark mode
- 🌐 Web access

## 🤖 Supported Bots

- ChatGPT
- Google Bard
- Claude 3.5



## 🔨 Build from Source

- Clone the source code
- `corepack enable`
- `yarn install`
- `yarn build`
- In Chrome/Edge go to the Extensions page (chrome://extensions or edge://extensions)
- Enable Developer Mode
- Drag the `dist` folder anywhere on the page to import it (do not delete the folder afterward)

- データ収集について：
  - 公式エクステンションストアでの配布時は、ストア標準の統計情報（インストール数等）以外のデータは一切収集いたしません

### English
Our privacy policy is as follows:

- Data Collection:
  - When distributed through official extension stores: We only collect standard store metrics (e.g., installation numbers)
  - When installed from GitHub: No data collection whatsoever
- Data Storage:
  - All settings and chat histories are stored exclusively in your browser's local storage
  - Depending on your browser settings, this data may be synchronized to your browser vendor's servers through the browser's built-in sync functionality
- AI API Usage:
  - The use of AI chat service APIs configured by users is outside the scope of this privacy policy
  - Users are responsible for their own API usage
